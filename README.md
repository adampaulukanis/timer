# timer

I need a timer.

I could of course use a pomodoro app or any other timer, but I want to make my
own version of this. And here it is.

![My super app seen on different browsers!](./mytimer1.png "my app")

## Url

Check [this](https://adampaulukanis.github.io/timer/) out.

## What is wrong

- When you're using Safari make sure you have
  Preferences->Websites->Auto-Play->Allow All Auto-Play On! Or my timer will not
  work.
